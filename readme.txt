Wrapper around Social Links block to make them collapsible.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wi-collapsible-social-links` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
